📝 To-Do App

A simple and responsive To-Do List web application built using HTML, CSS, and JavaScript. It helps users manage daily tasks efficiently with an easy and clean interface.

🚀 Features
➕ Add new tasks
✔️ Mark tasks as completed
🗑️ Delete tasks
🎯 Simple and clean UI
📱 Responsive design
🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
📂 Project Structure
/todo-app
│── index.html
│── style.css
│── script.js
│── README.md
💡 How It Works
User enters a task in the input box
Clicks the add button to save it
Tasks can be marked complete or deleted
All interactions are handled using JavaScript DOM manipulation
📌 Purpose

This project was built to improve core frontend development skills such as DOM manipulation, event handling, and responsive UI design.
